# Local Agent Memory

This directory is for local-only private notes attached to named agents.

Keep it out of version control and never store secrets here unless local policy explicitly permits it.

Recommended use:
- temporary repo observations not ready for shared memory
- local operator preferences, scratch checklists, or machine-specific reminders
- short-lived notes that should not become part of the committed framework

Mirror the shared agent names here when a local-only memory surface is useful.

Audit-only agents can use this space for temporary triage notes before findings are turned into shared issues.
