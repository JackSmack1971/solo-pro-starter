# Loop Prompt

Run one focused maintenance iteration for this repository.

Constraints:

- Start by reading project instructions and current git state.
- Prefer one verified improvement over many speculative edits.
- Treat deployments, signer changes, ABI drift, workflow edits, and chain config as high-risk.
- Stop after one completed and verified improvement or one confirmed blocking finding.

Required output:

- chosen scope
- evidence read
- action taken
- verification performed
- remaining risk
